**Mineral Frontend**

---

1. Clone this project
2. Run `npm i` on terminal. This command will download all required node_modules.
3. Run `npm run start`.
4. Open http://localhost:3000

---

### Watch
Run `npm run watch`

Run `npm run build` when you ready to staging or production to the server.
